# pyray
Resources for ray tracing in python with POV-Ray.

![](spheres.png)

Ray traced with povray using [vapory](https://github.com/Zulko/vapory). Script available [here](https://gist.github.com/ronojoy/bd6c15603f7f39c6d10d)
